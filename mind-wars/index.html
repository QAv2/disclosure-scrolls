The Program

Project MKULTRA

In 1953, CIA Director Allen Dulles authorized a program to explore “the use of biological and chemical materials in altering human behavior.” It would run for 20 years across 149 sub-projects at 80+ institutions — universities, hospitals, prisons, military bases. Its name was MKULTRA.

The subjects were often unwitting. The methods included LSD, electroshock, sensory deprivation, hypnosis, and psychological torture. The goal was control — not understanding. Not healing. Control.

Documented
MKULTRA encompassed 149 sub-projects across 80+ institutions including Harvard, Stanford, Columbia, McGill, and the National Institute of Mental Health. Methods included LSD administration to unwitting subjects, electroshock, sensory deprivation, isolation, verbal and sexual abuse, and hypnosis. The program ran from 1953 to 1973.
Senate Select Committee on Intelligence, “Project MKULTRA” (1977). Church Committee Final Report, Book I (1976). CIA admission to Congress.
Documented
In 1973, CIA Director Richard Helms ordered all MKULTRA files destroyed. The directive was carried out. But 20,000 pages survived — only because they had been misfiled in the CIA’s financial records division and were discovered during a 1977 FOIA request by investigative journalist John Marks. These pages became the basis for the Church Committee investigation and every public disclosure that followed.
Senate Select Committee hearings (1977). John Marks, The Search for the “Manchurian Candidate” (1979). CIA Inspector General Report (1963, declassified).

The Weapons

The Frey Effect

Documented
The microwave auditory effect was first reported in peer-reviewed literature in 1961 by neuroscientist Allan Frey in the journal Aerospace Medicine. Modulated microwave radiation at specific frequencies produces sounds perceived inside the head — clicks, buzzes, even words — with no external speaker, no earpiece, and no physical contact. The effect has been independently replicated in laboratories worldwide for over 60 years.
Allan Frey, “Auditory System Response to Radio Frequency Energy,” Aerospace Medicine 32 (1961). Subsequent replication: Lin & Wang (2007), Elder & Chou (2003), Foster & Finch (1974).

Documented

The State Department kept the irradiation secret from embassy staff for years and commissioned a classified study by Johns Hopkins University to assess the effects.
Johns Hopkins “Moscow Signal” study (declassified). Declassified State Department cables. National Security Archive. AP and UPI reporting (1976).

The Suppression

In 1995, the CIA commissioned an external review of the Stargate program by the American Institutes for Research. The review acknowledged statistically significant results — then recommended termination anyway, citing lack of “actionable intelligence.” The program was declared dead. #!/usr/bin/env python3
"""
Extract files hidden in a CNTYSEED steganographic image.
Only requires: pip install Pillow
Usage: python3 extract.py <image.png> <output_dir>
"""
import hashlib, io, os, struct, sys, tarfile, zlib
from PIL import Image

MAGIC = b"CNTYSEED"
HDR_SIZE = 64
HDR_FMT = "!8sBBBBII32sHH8s"


def lsb_decode(flat, depth, num_bytes):
    total_bits = num_bytes * 8
    pad = depth - (total_bits % depth) if total_bits % depth else 0
    slots = (total_bits + pad) // depth
    mask = (1 << depth) - 1
    bits = []
    for i in range(slots):
        val = flat[i] & mask
        for shift in range(depth - 1, -1, -1):
            bits.append((val >> shift) & 1)
    out = bytearray()
    for i in range(0, num_bytes * 8, 8):
        byte = 0
        for b in range(8):
            byte = (byte << 1) | bits[i + b]
        out.append(byte)
    return bytes(out)


def extract(image_path, output_dir):
    img = Image.open(image_path).convert("RGB")
    flat = [ch for px in img.getdata() for ch in px]

    hdr = None
    for depth in (1, 2, 3, 4):
        raw = lsb_decode(flat, depth, HDR_SIZE)
        if raw[:8] != MAGIC:
            continue
        fields = struct.unpack(HDR_FMT, raw)
        if fields[3] == depth:
            hdr = {
                "flags": fields[2], "lsb_depth": depth,
                "compression": fields[4], "original_size": fields[5],
                "compressed_size": fields[6], "sha256": fields[7],
            }
            break

    if not hdr:
        print("No embedded data found.")
        sys.exit(1)

    if hdr["flags"] & 0x01:
        print("This image is encrypted.")
        sys.exit(1)

    print(f"Found: {hdr['original_size']:,} bytes, {hdr['lsb_depth']}-LSB")

    total = HDR_SIZE + hdr["compressed_size"]
    raw = lsb_decode(flat, hdr["lsb_depth"], total)
    payload = raw[HDR_SIZE:]

    if hdr["compression"] == 1:
        payload = zlib.decompress(payload)
    elif hdr["compression"] == 2:
        import lzma
        payload = lzma.decompress(payload)

    if hashlib.sha256(payload).digest() == hdr["sha256"]:
        print("Integrity: VERIFIED")
    else:
        print("WARNING: SHA-256 mismatch")

    os.makedirs(output_dir, exist_ok=True)
    if hdr["flags"] & 0x02:
        with tarfile.open(fileobj=io.BytesIO(payload), mode="r") as tar:
            tar.extractall(path=output_dir, filter="data")
        print(f"Extracted to: {output_dir}/")
    else:
        with open(os.path.join(output_dir, "extracted"), "wb") as f:
            f.write(payload)
        print(f"Written to: {output_dir}/extracted")


if __name__ == "__main__":
    if len(sys.argv) != 3:
        print(f"Usage: {sys.argv[0]} <image.png> <output_dir>")
        sys.exit(1)
    extract(sys.argv[1], sys.argv[2])
